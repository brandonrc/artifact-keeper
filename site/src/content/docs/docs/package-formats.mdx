# Package Formats

Artifact Keeper supports **45+ package formats** out of the box. Every format is included at no extra cost — no feature gates, no tier restrictions. Each speaks the native wire protocol that your package manager already expects.

This page is a quick reference index. Click through to the detailed guide pages for configuration examples, native client setup, and best practices.

---

## Custom Formats

Artifact Keeper can be extended with custom format handlers using the [WASM plugin system](/docs/advanced/plugins/). Write handlers in any language that compiles to WASM (Rust, Go, C, AssemblyScript) and install via Git URL, ZIP upload, or local path.
